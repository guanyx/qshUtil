#企商汇常用工具集合

企商汇常用工具集合

调用方式

工具集合作为全站通用依赖已经更新到qsh-resource
```js
//如果没有安装qsh-resource
npm install qsh-resource --save-dev
//如果已经安装过qsh-resource
npm update
//qsh-resource更新了iconfont，具体细节查看iconfont的ReadMe
```


```js
qshUtil.spinner(options)    //加载进度条
qshUtil.header(options)     //移动端页面通用头部
qshUtil.module(options)     //模块加载
```
